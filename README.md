# archibe-backend

## Develop

### Start database

```bash
$ make startdb
```

### Stop database

```bash
$ make stopdb
```

### Init database

```bash
$ make initdb
```

### Insert YouTube archive data

```bash
$ CHANNEL_ID=<channel-id> make insert
```

### Start Server

```bash
$ make serve
```

### Test

```bash
$ make test
```
